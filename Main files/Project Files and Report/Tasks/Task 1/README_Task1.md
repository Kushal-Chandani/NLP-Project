
# Task1_Scores

This repository contains a Jupyter Notebooks for computing and analyzing different model scores for text data in a different context as part of an NLP Task. The notebook includes:

- **Data Preprocessing:** Text cleaning and preparation.
- **Model Score Computation:** Calculate similarity scores using the BART model.
- **Analysis and Insights:** Visualize and interpret score trends.
- **Performance Metrics:** Evaluate text generation models.

## Requirements

- Python 3.7+
- Libraries: `transformers`, `torch`, `pandas`, `matplotlib`, `seaborn`, `nltk`.

## Usage

1. Clone the repository.

2. Install the required dependencies.

3. Open the notebook and run the cells.


## Output

- Model scores for text pairs.
- Graphical insights from score analysis.
- Optionally export results in CSV format.
